Surgimento da internet

Voltar

A internet surgiu na época da guerra fria através da necessidade de se fazer cálculos complexos e comunicação, afim de facilitar as estratégias de guerra. Nesse momento, surge o protótipo da primeira rede de internet, a Arpanet.

"Ligada por um backbone (“espinha dorsal”, isto é, estruturas de rede capazes de manipular grandes volumes de informações) que passava por debaixo da terra, o que dificultava sua destruição. O acesso à ARPANET era restrito a militares e pesquisadores, demorou chegar ao público em geral, pois temiam o mau uso da tecnologia por civis e países não-aliados."

Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (WWW). Primeiro navegador chamado WorldWideWeb.

Curiosidade:Tim Berners-Lee é considerado o pai da internet e em 6 de agosto de 1991 ele publicou a primeira página WEB, e ainda está lá até hoje.

História dos computadores

Voltar

1°Geração:

  • MARK 1, um dos primeiros computadores, 1946.
  • COLOSSUS, criado pela alemanha, 1946.
  • ENIAC, criado pelos EUA. Usado para calcular rotas balísticas.

2°Geração:

Ocorreu de 1959 a 1965. Foi a evolução das válvulas para transitores, isso permitiu que os computadores diminuissem bastante o seu tamanho.

3°Geração:

Ocorreu de 1965 a 1970. Foi criado os circuitos integrados: Transistor, Capacitores, Resistores, assim dimuindo mais ainda o tamanho dos computadores.

4°Geração:

A partir de 1971. Circuitos integrados passaram para micro processadores. OBS: Foi nesta geração que foi criado o mouse pela APPLE.

O que são Clients

Voltar

Cliente: Em uma infraestrutura cliente-servidor, cliente é um dispositivo de computação, como desktops, laptops, smartphones, tablets ou aplicativos que, através de protocolos de rede se conectam a servidores para fazer uma solicitação de serviço ou recurso e aguarda a resposta.

O que são Servers

Voltar

Um server é um computador ou software mais potente que um desktop comum. Ele é capaz de conter um ou mais processadores, além de bancos de memórias, portas de comunicação, diferentes softwares e sistemas de armazenamento de dados.

Alguns exemplos de Tags HTML:

Voltar
  1. Tag (p): É utilizada para maracações de parágrafos.
  2. Tag (font): Era utilizada nas versões antiga do HTML, não suportada no HTML5, toda modificação deve ser feita através do CSS.
  3. Tag (h1 até h6): São tags geralmente usadas para título.
  4. Tag (mark): Utilizada para marcar texto.
  5. Tag (del): Usada para definir um texto deletado de um documento.
  6. Tag (abbr): Usada para definir abreviações ou acronimos.
  7. Tag (a): É utilizada para links de navegação seja na mesma página ou ir para outras páginas.
  8. Tag (u): Utilizada para sublinhar
  9. Tag (blockquote): Usada para citações, frases de outro autor.
  10. Tag (small): Usada para definir copy-right e alguns comentários.
  11. Tag (i): Usada para deixar o texto em itálico.
  12. Tag (strong): Usada para deixar o texto em negrito.
  13. Tag (ol): Utilizada para fazer listas ordenadas.
  14. Tag (ul): Usada para listas desordenadas.
  15. Tag (li): Utilizada para indicar uma lista.
  16. Tag (hr): É usada geralmente em de forma horizontal, usando uma linha para separar determinado texto, ela cria uma linha na horizontal.
  17. Tag (sub): Usada para definir subscript, como por exemplo uma fórmula química na mesma linha.
  18. Tag (sup): Utilizada para superscript, ou footnote(notas de rodapé).
